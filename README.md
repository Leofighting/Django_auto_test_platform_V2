### 平台介绍

> 支持 `API、AppUI、WebUI` 等自动化测试，集成实现测试用例管理、产品管理、任务计划、测试报告、定时任务或持续集成等功能模块。使自动化测试的过程达到数据、脚本、任务分离易于维护和管理，成本更低、数据更直观、产出更快等目标。
